# Freelancer Client Workflow

## The 5 Stages

### Stage 1: Paid
**Goal:** Collect materials, set expectations

**Actions:**
1. Send welcome email with checklist
2. Add client to dashboard (stage: 1)
3. Request domain access or purchase
4. Set deadline: 7-10 days from materials received

**Checklist:**
- [ ] Payment confirmed
- [ ] Welcome email sent
- [ ] Materials requested
- [ ] Project added to dashboard

---

### Stage 2: In Review
**Goal:** Gather all materials before starting work

**Actions:**
1. Review received materials
2. Clarify any missing items
3. Confirm design direction
4. Update checklist items as received

**Checklist:**
- [ ] Domain access confirmed
- [ ] Content/copy received
- [ ] Images/photos received
- [ ] Design preferences discussed

---

### Stage 3: Design
**Goal:** Build the site

**Actions:**
1. Create first draft
2. Don't show client yet (avoid premature feedback)
3. Complete all pages
4. Test on mobile

**Checklist:**
- [ ] All pages built
- [ ] Mobile responsive
- [ ] Links working
- [ ] Images optimized

---

### Stage 4: Review
**Goal:** Get client feedback, make revisions

**Actions:**
1. Send preview link
2. Request specific feedback (not "let me know what you think")
3. Track revision rounds (2 included)
4. Update dashboard with feedback notes

**Checklist:**
- [ ] Preview link sent
- [ ] Feedback received
- [ ] Revisions complete (round 1)
- [ ] Revisions complete (round 2 - if needed)

---

### Stage 5: Launch
**Goal:** Go live, hand off ownership

**Actions:**
1. Final review with client
2. Deploy to production
3. Configure domain DNS
4. Send login credentials securely
5. Deliver documentation

**Checklist:**
- [ ] Client final approval
- [ ] Site deployed live
- [ ] Domain connected
- [ ] Credentials sent
- [ ] Documentation delivered
- [ ] Support window started (30 days)

---

## Demo Lead Workflow

### Finding Leads
1. Search social media for "need a website"
2. Check Google Maps for businesses without sites
3. Ask existing clients for referrals

### Outreach
1. Build quick demo (1-2 hours max)
2. Send personalized email with demo link
3. Log in dashboard as "building" status
4. Follow up in 3 days if no response

### Conversion
1. If interested → move to Stage 1 (Paid)
2. If not now → keep in dashboard, follow up monthly
3. If declined → mark as declined, move on

---

## Daily Habits

**Morning (5 mins):**
- Check dashboard for overdue items
- Review Stripe for new payments
- Check if any demos need follow-up

**End of day (5 mins):**
- Update client checklists
- Log any conversations in notes
- Check revenue totals

**Weekly (30 mins):**
- Review all active projects
- Send follow-ups to old leads
- Plan demo builds for next week

---

## Red Flags

**Watch for:**
- Client won't pay deposit → Don't start work
- No materials after 2 weeks → Pause project
- Scope creep ("can you just add...") → Charge more or push back
- Vague feedback ("make it pop") → Ask for examples

**Your move:**
- Be professional but firm
- Document everything in dashboard notes
- It's okay to fire bad clients
